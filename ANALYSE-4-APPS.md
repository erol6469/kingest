# Analyse des 4 Meilleures Apps de Trading — Ce qu'on prend pour KINGEST

---

## 1. eToro — Le Social Trading King

### Ce qu'on PREND (le bon)
- **CopyTrader** : copier automatiquement les trades d'autres investisseurs (déjà esquissé dans Kingest)
- **Sentiment du marché** : indicateur haussier/baissier communautaire sur chaque actif
- **Smart Portfolios** : investir dans des paniers thématiques (Tech, Green Energy, Gaming...)
- **Compte démo $100K** : mode simulation pour s'entraîner sans risque
- **Intérêts sur le cash** : afficher un rendement sur le cash non investi (3.8%)
- **Actions fractionnées** : investir dès $10 dans n'importe quelle action

### Ce qu'on JETTE (le mauvais)
- Frais de retrait $5
- Frais de conversion devises élevés
- Pas de compte retraite
- Support client lent
- Charting mobile limité

---

## 2. Trading 212 — Le Zero Commission

### Ce qu'on PREND (le bon)
- **0% commission** : aucun frais sur actions et ETF
- **7,000+ actions** sur les plus grandes bourses mondiales
- **Multi-devises** : GBP, USD, EUR, CHF, etc. — pas de frais de conversion si même devise
- **AutoInvest / Pies** : créer des "tartes" (portfolios personnalisés) avec allocation automatique
- **Intérêts cash élevés** : 4.68% sur le cash non investi
- **Actions fractionnées** ultra-faciles

### Ce qu'on JETTE (le mauvais)
- Pas de fonds mutuels
- Spreads forex élevés
- Charting trop basique
- Pas de copy trading
- Support lent

---

## 3. Revolut — Le All-in-One Fintech

### Ce qu'on PREND (le bon)
- **Interface ultra-clean** : design minimaliste, navigation fluide, UX parfaite
- **Achats récurrents** : investir automatiquement chaque semaine/mois (DCA)
- **Robo-advisor** : portfolio automatisé selon profil de risque
- **Staking crypto** : gagner des rendements sur ses cryptos
- **Alertes de prix** : notifications push personnalisées
- **Transferts de portfolio** : transférer ses positions entre plateformes

### Ce qu'on JETTE (le mauvais)
- Nombre limité d'actions (3,000 vs 7,000+)
- Outils avancés manquants
- Support client problématique
- Frais de conversion sur plan gratuit

---

## 4. Interactive Brokers (IBKR) — Le Pro Global

### Ce qu'on PREND (le bon)
- **170+ marchés mondiaux** : bourses de Paris (CAC 40), Tokyo (Nikkei), Singapour (SGX), Londres (FTSE), Hong Kong (HSI), etc.
- **Sélecteur de bourse** : choisir la bourse exacte où acheter une action
- **Indices investissables** : CAC 40, S&P 500, Nikkei 225, DAX, FTSE 100 via ETF
- **Scanner de marché** : filtrer par secteur, pays, capitalisation, dividende...
- **Ordres avancés** : limit, stop, trailing stop, bracket orders, OCO
- **Multi-devises natif** : trader dans la devise locale de la bourse
- **Recherche intégrée** : analyses fondamentales, ratios P/E, dividendes
- **AI news summaries** : résumés d'actualités par IA

### Ce qu'on JETTE (le mauvais)
- Interface complexe pour débutants
- Courbe d'apprentissage raide
- Frais d'inactivité (supprimés récemment mais historique)
